Informações da API

Endpoint de Status: GET /api/status
Endpoint de Envio: POST /api/send
Autenticação: Authorization: Bearer <API_TOKEN> 
curl -X POST /api/send \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <API_TOKEN>" \
  -d '{
  "to": "5511999999999",
  "type": "text",
  "text": "Olá, tudo bem?"
}'
