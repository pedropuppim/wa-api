Documentação da API

Autenticação: Authorization: Bearer <API_TOKEN>
GET /api/status

Retorna o status da conexão do WhatsApp

Resposta: 
{"status": "READY", "qrAvailable": false}
POST /api/send

Envia uma mensagem de texto

Parâmetros:
  • to - Número do destinatário
  • type - "text"
  • text - Conteúdo da mensagem
curl -X POST /api/send \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <API_TOKEN>" \
  -d '{
  "to": "5511999999999",
  "type": "text",
  "text": "Olá, tudo bem?"
}'
POST /api/send

Envia uma imagem via URL

Parâmetros:
  • to - Número do destinatário
  • type - "image"
  • caption - Legenda (opcional)
  • image.source - "url" ou "base64"
  • image.url - URL da imagem
curl -X POST /api/send \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <API_TOKEN>" \
  -d '{
  "to": "5511999999999",
  "type": "image",
  "caption": "Legenda opcional",
  "image": {
    "source": "url",
    "url": "https://exemplo.com/imagem.jpg"
  }
}'
POST /api/send

Envia um áudio (como mensagem de voz)

Parâmetros:
  • to - Número do destinatário
  • type - "audio"
  • audio.source - "url" ou "base64"
  • audio.url - URL do áudio
  • asPtt - true para mensagem de voz
curl -X POST /api/send \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <API_TOKEN>" \
  -d '{
  "to": "5511999999999",
  "type": "audio",
  "audio": {
    "source": "url",
    "url": "https://exemplo.com/audio.mp3"
  },
  "asPtt": true
}'
POST /api/pause

Pausa o webhook para um contato

Parâmetros:
  • phone - Número do contato
curl -X POST /api/pause \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <API_TOKEN>" \
  -d '{
  "phone": "5511999999999"
}'
