Documentação da API

Autenticação: Authorization: Bearer <API_TOKEN>
GET /api/status Retorna o status da conexão do WhatsApp

Resposta

{"status": "READY", "qrAvailable": false, "lastError": null}

Status possíveis

DISCONNECTEDDesconectado
CONNECTINGConectando
QR_REQUIREDAguardando QR Code
READYPronto para enviar
POST /api/send Envia uma mensagem (texto, imagem ou áudio)

Parâmetros do Body

CampoTipoObrigatórioDescrição
tostringSimNúmero do destinatário (ex: 5511999999999)
typestringSim"text", "image" ou "audio"
textstringSe type=textConteúdo da mensagem de texto
captionstringNãoLegenda para imagem
imageobjectSe type=imageObjeto com dados da imagem
image.sourcestringSim"url" ou "base64"
image.urlstringSe source=urlURL da imagem
image.datastringSe source=base64Dados em base64
image.mimetypestringSe source=base64Ex: "image/jpeg"
audioobjectSe type=audioObjeto com dados do áudio
audio.sourcestringSim"url" ou "base64"
audio.urlstringSe source=urlURL do áudio
asPttbooleanNãotrue para enviar como mensagem de voz

Exemplos de Request

curl -X POST /api/send \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <API_TOKEN>" \
  -d '{
  "to": "5511999999999",
  "type": "text",
  "text": "Olá, tudo bem?"
}'
curl -X POST /api/send \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <API_TOKEN>" \
  -d '{
  "to": "5511999999999",
  "type": "image",
  "caption": "Legenda opcional",
  "image": {
    "source": "url",
    "url": "https://exemplo.com/imagem.jpg"
  }
}'
curl -X POST /api/send \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <API_TOKEN>" \
  -d '{
  "to": "5511999999999",
  "type": "audio",
  "audio": {
    "source": "url",
    "url": "https://exemplo.com/audio.mp3"
  },
  "asPtt": true
}'

Resposta de Sucesso

{"ok": true, "id": "ABCD1234", "to": "5511999999999@c.us", "type": "text"}
POST /api/pause Pausa o webhook para um contato

Parâmetros do Body

CampoTipoObrigatórioDescrição
phonestringSimNúmero do contato a pausar
 
curl -X POST /api/pause \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <API_TOKEN>" \
  -d '{
  "phone": "5511999999999"
}'

Resposta de Sucesso

{"ok": true, "message": "Contact paused for 4 hours", "data": {"chatId": "5511999999999@c.us", "phone": "5511999999999", "expiresAt": 1737424400000}}
